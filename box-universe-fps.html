Box Universe Slicer 3D

Explore generalized Collatz sequences in a 3D "Box Universe."

Generalized Collatz Rules:

For a number $n$:

  • If $n$ is divisible by X, then $n \rightarrow n / X$
  • Else, $n \rightarrow (Y \times n) + Z$

N (Base) defines the size of the cubic grid (N x N x N). Each cube represents a number from $0$ to $(N^3 - 1)$.
