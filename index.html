Collatz Box Universes

View on GitHub Status: Disabled

Explore and visualize generalized Collatz sequences to discover hidden patterns, foster intuition, and gain new perspectives on dynamical systems.

Rule: If $n \pmod{X} = 0$, then $n \to n / X$. Otherwise, $n \to n \cdot Y + Z$.

(Standard Collatz uses X=2, Y=3, Z=1)
